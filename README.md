# Java Quiz

## Pre-Requisites 
 - JRE & JDK should be installed
 - Basic Knowledge of Java Swing and AWT

## Library used to make UI:
 - Swing
 - AWT

## How to run:
 - ``javac gui.java``
 - ``java quiz.java``
